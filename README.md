# frete-correios-js
    Calculo para frete via correios buscando os dados na API dos correios,
    usando o pacote node-correios.
    Desenvolvido em NodeJS e ReactJS.

# Passos para instalação
    Clonar via GitHub.

# Abrir terminal 1
    cd back
    npm install

# Abrir terminal 1
    cd front
    npm install

# Rodar os servidores
    em ambos os terminais executar: npm start
